# 🐍 Algoritmos e Estruturas de Dados em Python

Este repositório documenta minha jornada de aprendizado em algoritmos, lógica de programação e estruturas de dados. 

O foco inicial é a implementação prática dos conceitos apresentados no livro **"Entendendo Algoritmos" (Grokking Algorithms)**, de Aditya Y. Bhargava, utilizando **Python**.

## 📚 Sobre o Projeto
O objetivo não é apenas copiar o código do livro, mas:
1.  **Implementar** os algoritmos "do zero".
2.  **Testar** casos de borda e variações.
3.  **Documentar** a complexidade (Big O) de cada solução.

Futuramente, este repositório também servirá para armazenar soluções de desafios de plataformas como *LeetCode* e *HackerRank*.
